# decoterm
ANSI Escape Sequences Helper  
Decorate + Terminal = decoterm!

# how to use
Simple!  
decoterm(*args, options, **kwargs)  
  
options:  
- FGColors            ... Set ForeGround Colors
- BGColors            ... Set BackGround Colors
- FontStyle           ... Set Font Style
- SetFG8bit(n)        ... Set ForeGround Colors (8bit)
- SetBG8bit(n)        ... Set BackGround Colors (8bit)
- SetFG24bit(r, g, b) ... Set ForeGround Colors (24bit)
- SetBG24bit(r, g, b) ... Set BackGround Colors (24bit)
